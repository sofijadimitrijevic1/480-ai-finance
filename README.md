AI Financial Fraud Detection

This is a project for Rodrigo Canaan's CSC 480 class at Cal Poly SLO. 
Code created by Sofija Dimitrijevic, Karen Severson, Sabrina Huang, Glenn Carvalho, and Anissa Soungpanya. We used PyTorch to create our neural models.

How to Run:
In terms of dependecies, make sure that numpy, sklearn, pandas, and torch are all installed on your computer.

preprocessing.py contains the data preprocessing code, run this first. The main code can be run in model_comparison.ipnyb just run each cell sequentially. The grad_cam file contains code that was created to attempt using GradCam, can be added into a cell in model_comparison. model_comparison will have all the main results shown in our report and presentations.
